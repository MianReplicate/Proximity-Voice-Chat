# Love Voice Chat Simulator? Wish it could be proximity instead?

Well this is the mod for you! Introducing Proximity Voice Chat... oh no.. your poor ears!

## What does Proximity Voice Chat do?
- Bots can now chatter all around you! (Configurable Time)
    - Oneliners (Bot says one thing)
    - Conversations (Bots talk back and forth) 
    - Due to how this is setup, it is possible to make your own addon to add extra voicelines! There is no documentation at the moment, but if you are a coder, the process would be very easy if you wanted to make an addon.
- Upon death, bots will give you ear damage! [(Requires Voice Chat Simulator)](https://steamcommunity.com/sharedfiles/filedetails/?id=2937907325)
- Lots and lots of configuration options to fit every experience!!
